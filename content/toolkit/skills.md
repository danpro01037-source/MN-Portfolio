# Technical Skills

This section outlines the various technologies and tools I'm proficient in, categorized by domain. These skills have been developed through professional experience, personal projects, and continuous learning.

## Frontend Development

My frontend skills focus on creating responsive, intuitive user interfaces with modern JavaScript frameworks including React, Next.js, Vue.js, and Angular. I specialize in TypeScript, JavaScript, and modern CSS frameworks like Tailwind CSS and Bootstrap.

## Mobile Development

I have extensive experience building cross-platform mobile applications using React Native and Flutter, targeting both iOS and Android platforms with optimized performance and native-like experiences.

## Backend Development

On the backend, I specialize in building scalable, maintainable services and APIs using Python (Django, FastAPI), Laravel (PHP), Node.js (Express), and C#/ASP.NET. I design REST and GraphQL APIs with proper authentication, authorization, and business logic.

## AI, Chatbots & Voice Agents

I develop AI-powered chatbots and conversational agents, integrating LLM APIs and building RAG pipelines for semantic search. I work with voice agent platforms like Retell and Vapi, implementing STT/TTS pipelines and prompt engineering for production applications.

## Web3 & Blockchain Development

I build Web3 features for decentralized and blockchain-enabled products, including wallet authentication, token logic, and smart contract interactions. I’m comfortable working with Solidity smart contracts and common toolchains like Hardhat and Ethers.js.

## Databases & Infrastructure

I work with relational databases (PostgreSQL, MySQL), NoSQL solutions (Firebase, Supabase), and vector databases (FAISS, Pinecone) for AI applications. I have experience with database design, optimization, and integration.

## DevOps, Cloud & Automation

I have extensive experience with Docker containerization, Kubernetes orchestration, AWS cloud services (EC2, S3, Lambda), and CI/CD pipelines (GitHub Actions, Vercel). I design and implement automation workflows using n8n, integrating CRMs, databases, and third-party platforms.

## Tools & Collaboration

I'm proficient with version control (Git, GitHub), development environments, and collaboration tools. I work effectively in Agile/Scrum environments and maintain clear technical documentation.
