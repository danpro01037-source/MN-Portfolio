## Project Overview

Loyal is a relationship health quiz application that uses LoyalAI to help users explore their relationship dynamics and discover a personalized love score. The platform delivers an engaging, interactive experience focused on relationship insights and user reflection.

## Project Details

### Technologies Used

* AI / LoyalAI integration
* Interactive quiz flows
* Modern web application stack

### Key Features

* **Relationship health quiz**: Guided questions to assess and reflect on relationship dynamics
* **Love score**: AI-powered, personalized love score based on user responses
* **Engaging UX**: Clean, user-friendly interface for an emotional and reflective experience

## Portfolio Highlights

* **AI-driven consumer app**: Combining quiz UX with AI-generated insights
